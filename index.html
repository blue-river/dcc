    DCC

    DCC is a fast optimizing compiler for the DC language, targeting the DCPU-16. DC is a language with a simple C-like syntax.

    A DC program consists of the .dc source files in the current directory, along with source files in any specified module search path. Each .dc source file contains the code for the module with the same name.

    This is a simple DC program showing the language syntax:

    int foo = 4;
int answer;

void main()
{
    int bar;

    bar = 3;

    while (foo != 2)
    {
        foo = bar;
        bar--;
    }

    doTheFoo(foo);

    setAnswer(&answer);
}

void doTheFoo(int bar)
{
    foo = bar + 2;
}

void setAnswer(int answer)
{
    *answer = 42;
}

    DC language

    The basic unit of compilation is a module. Compilation starts with the main module, which must contain the main function.

    To access methods from another module, prefix their name with the module name and a period.

    Dependencies

    DCC requires Python 2.7 and PLY. PLY can be installed using easy_install ply.

    Add the DCC directory to the PATH to use dcc.py easily in any directory.
